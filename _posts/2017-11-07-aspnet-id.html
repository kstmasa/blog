 Git 到本機端時，發現會有這樣的Error，明明compile沒有問題，但就是會有這個Error


     後來經過Google後，發現是要把Project裡面的 .vs 資料夾 刪掉就可以佈署到IIS了

https://blog.miniasp.com/post/2015/05/12/ASPNET5-Developers-Note-Part-02-Project-Structure-and-xproj-file.aspx

     裡面有提到

  • .vs 隱藏資料夾：用來保存原本我們在目錄中會看到的 *.suo 檔案，這個檔案主要用來保存 Visual Studio 2015 操作過程的一些狀態，例如你已經開啟的檔案清單等暫存資訊。如果 ASP.NET 5 網站使用 IISExpress 進行偵錯，IISExpress 所需要的 applicationhost.config 設定檔也會放在 .vs\config\ 目錄下。 
    ※ 這個資料夾不應該加入版本控制。
     所以說，在Git上面，應該要ignore .vs 這個folder，這樣才不會出錯
